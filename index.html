Inernship with Saudi Aramco

Electrical Engineering trainee

for 7 monthes
June 2019 - January 2020

I have been trained in Saudi Aramco under supervision of Mr.Abdulrahman AL-Johanni and Mr. Mohammed AL-Salman in the Mechanical Southren Shops Department will known as the MSSD. from the University side I was supervised by Dr.Fahhad Alharbi.

In this job, I was trained on reparing induction motors from A to Z. That includes testing the motor preparing the coils and using the tools involved in the procedure. I was able to deliver an electric system to the company to be used in the winding machine to count the windings in both directions. I have participated in visits arranged by MSSD for the trainees to other departments such as the Aviation and Marine departments.

Featured Projects

I am one of those who believe strongly in learning by doing, but that does not mean you do not study the theory first. it is all about finding the correct balance between theory and practice. here you will see some of the projects I have made using my knowledge in electrical engineering or tech in general.

Winding Machine Counter

This project is related to the winding machine we were using in Saudi Aramco. the problem we faced is that the machine can not count the number of windings in a coil properly. In other words, it can not deduct the reverse turns which was costing money and time. In this project I have used an Arduino Nano, IR sensors, coding skills, and some physics.Thankfully, the problem was solved and I have presented the system to the manager of the MSSD department. details

Raspberry pi Toy Car

this project is a project that I have done with a team of four students while in university. we have used a raspberry pi computer to control a toy car using smart phone. we were also able to make the car follow a line and avoid obsticals that was using IR sensors and ultrasonic sensors. details

Repurposing an IR remote control

this is a project I have done using an Arduino Microcontroller and any remote contrl that uses infrared signals. In this project, I was able to capture the signal from the remote control using an Arduino and use that signal in controling an LED (or can be any electric device) connected to the Arduino.
