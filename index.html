Winding Machine Counter

This project is related to the winding machine we were using in Saudi Aramco. the problem we faced is that the machine could not count the number of windings in a coil properly. In other words, it can not deduct the reverse turns which was costing money and time, and made the whole work harder for the employees to do. In this project I have used an Arduino Nano, IR sensors, coding skills, and some physics.Thankfully, the problem was solved and I have presented the system to the manager of the MSSD department during the -QSI- quarterly safety inspection. details
